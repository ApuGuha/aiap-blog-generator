# aiap-blog-generator
Requirements:
1.WordPress 6.0+
2.PHP 7.4+
3.OpenAI API Key
4. Internet access from the server

Download or clone this repository:
git clone https://github.com/your-username/ai-blog-generator.git

Upload the folder to: wp-content/plugins/
Go to WordPress Admin → Plugins
Activate AI Blog Generator

Configuration:
1. Go to WordPress Admin → AI Blog Generator
2. Enter your OpenAI API Key
3. Click Save

Usage:
1. Open AI Blog Generator in admin menu
2. Enter a blog topic
3. Choose post status (Draft / Publish)
4. Click Generate Blog
